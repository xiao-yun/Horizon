## [LG 显示器未经用户同意通过 Windows Update 静默安装软件](https://videocardz.com/newz/lg-monitors-silently-install-software-through-windows-update-without-user-consent) ⭐️ 8.0/10

当用户通过 HDMI 连接 LG 显示器时，Windows Update 会在后台自动下载并安装 LG 提供的第三方软件，整个过程无需任何用户交互且无提示，该软件具有完整的系统和网络访问权限。此情况也适用于已连接过的旧款 LG 显示器。 此行为构成严重的安全和隐私隐患，因为自动安装的软件不受沙盒限制且随系统启动，可能成为恶意攻击的入口。它揭示了 Windows 硬件驱动程序分发机制可能被滥用以推送非必要甚至危险的应用程序，影响广大用户。 安装的软件在系统启动时即运行，拥有完整系统权限且无隔离。可通过组策略或设备安装设置中的‘阻止自动下载与设备元数据关联的应用程序’选项来禁用此类自动安装。

hackernews · baranul · Jul 18, 10:21 · [社区讨论](https://news.ycombinator.com/item?id=48956688)

**背景**: Windows 通过 DCH（声明式组件化硬件）驱动程序框架支持硬件厂商在驱动包中附带应用程序。微软允许厂商提交这些软件包至 Windows Update，并根据策略自动推送给用户。当检测到新设备时，系统可能自动安装所有关联软件，而不仅限于硬件必需的驱动。用户可通过设备安装设置控制自动下载行为。

**背景**: Apache Iceberg 是一种开放式表格式，允许多个计算引擎同时安全地操作同一张表。REST 目录是 Iceberg 提供的一种标准 API，用于客户端通过 HTTP 通信来加载、管理表元数据。LoadTableResponse 是加载表时返回的响应对象，目前包含了表的元数据和配置等信息。资源级访问控制是一种安全机制，可以对单个资源（如表）设置精细的访问权限，常用于云平台上结合 IAM 策略实现更安全的数据访问。
